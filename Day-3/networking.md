1. ip

Used to view and manage network interfaces, IP addresses, and routes.
```
ip addr
ip route
```

 2. ping

Checks connectivity between two hosts.
```
ping google.com
```

3. ifconfig

Displays or configures network interfaces (older command).
```
ifconfig
```

4. netstat

Shows network connections, routing tables, and ports.
```
netstat -tulnp
```

5. ss

Modern replacement for netstat.
```
ss -tulnp
```

6.  traceroute

Shows the path packets take to reach a destination.
```
traceroute google.com
```

7. nslookup

Queries DNS to resolve domain names.
```
nslookup google.com
```

8. dig

Advanced DNS lookup tool.
```
dig google.com
```

9. curl

Used to transfer data from or to a server (API testing).
```
curl http://example.com
```

10. wget

Downloads files from the internet.
```
wget https://example.com/file.zip
```
