.bashrc is a configuration file that runs every time you start an interactive Bash shell. <br>

its like "Run these commands every time I open a shell.”

### Common Uses of `.bashrc`

**Set Environment Variables:**
```
export PATH=$PATH:/usr/local/bin
export EDITOR=vim
export CUDA_VISIBLE_DEVICES=0
```

#### 1. Set File Desriptors limit to permenant.
open `~/.bashrc` <br>
Add below lines,

```
ulimit -n 65535
ulimit -Hn 65535
ulimit -Sn 65535
```
If you want it will automatically added in new created user then add this lines into `/etc/skel/.bashrc`
